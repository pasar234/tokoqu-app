# Stok Pintar Manajemen

Aplikasi web untuk manajemen stok produk dengan kategori NARITA, VR, KUDUS, dan LAIN LAIN. Aplikasi ini adalah Progressive Web App (PWA) yang dapat diinstal di perangkat mobile dan desktop.

## Fitur Utama

- ✅ Tambah produk baru dengan kategori
- ✅ Kelola stok produk (tambah/kurang)
- ✅ Pencarian produk
- ✅ Backup dan restore data
- ✅ PWA support (dapat diinstal)
- ✅ Tema gelap/terang
- ✅ Offline capability
- ✅ Responsif untuk mobile dan desktop

## Cara Menggunakan di GitHub Pages

### 1. Buat Repository di GitHub
- Buat repository baru dengan nama `stok-manajemen-app`
- Upload semua file ke repository

### 2. Aktifkan GitHub Pages
- Buka repository di GitHub
- Klik **Settings** → **Pages**
- Di bagian **Source**, pilih **main branch**
- Klik **Save**
- Tunggu beberapa menit sampai situs aktif

### 3. Akses Aplikasi
Aplikasi akan tersedia di: `https://[username-github].github.io/stok-manajemen-app/`

## Cara Install sebagai Aplikasi

### Di Desktop (Chrome/Edge):
1. Buka aplikasi di browser
2. Klik ikon **Install** di address bar
3. Klik **Install Stok Pintar**

### Di Mobile (Android/Chrome):
1. Buka aplikasi di Chrome
2. Klik menu (⋮) → **Add to Home screen**
3. Klik **Add**
